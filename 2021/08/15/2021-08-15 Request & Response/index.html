Http status code 狀態碼

用數字表示 reponse 的狀態，通常以開頭的數字做判斷
1xx ：稍等

2xx：成功

3xx：重新導向，用戶端瀏覽器必須採取更多動作才能完成要求。

4xx：Client 端錯誤

5xx：Service 端錯誤
